# Introduction-to-Supervised-Learning

# Supervised Learning Algorithms

## Project Overview
This project demonstrates the implementation of three supervised learning algorithms using Python and Scikit-learn:
- Linear Regression
- Logistic Regression
- Decision Tree Classifier

The project includes data preprocessing, model training, prediction, evaluation, and comparison of model performance.

## Datasets
### California Housing Dataset
- Used for Linear Regression (Regression Task)
- Source: Scikit-learn built-in dataset

### Iris Dataset
- Used for Logistic Regression and Decision Tree (Classification Task)
- Source: Scikit-learn built-in dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Evaluation Metrics

### Linear Regression
- Mean Squared Error (MSE)
- R² Score

### Classification Models
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Results
- Successfully implemented and evaluated Linear Regression, Logistic Regression, and Decision Tree algorithms.
- Compared the performance of classification models using standard evaluation metrics.
